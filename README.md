# YHMedicarePaySDK

移动医保结算

## Installation

YHMedicarePaySDK is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'YHMedicarePaySDK'
```
